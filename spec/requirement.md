# プロジェクト仕様
## プロジェクト概要
本プロジェクトはYahoo!路線情報で公開されている運行情報から、指定された路線の運行情報の内容を取得するプログラムです。

## 使用技術要素
本プロジェクトでは下記の技術要素を使用します。
- プログラム言語 ... Pytnon
- 使用ライブラリ ... BeautifulSoup4
- 検証環境 ... Docker

## プロジェクト詳細
本プロジェクトで運行情報を取得する対象の路線は下記のとおりです。
- 宇都宮線[東京～宇都宮]
- 宇都宮線[宇都宮～黒磯]
- 東京メトロ銀座線

上記の対象路線と対応するURLを管理するためのデータ構造をPythonプログラム内で定義し、上記の対象路線すべてに対して情報を取得します。
取得した情報は、路線名、状況、詳細の3つのデータをまとめた構造体として返します。
